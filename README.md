# 鼠标助手标准版 (MouseAssistantStd)

## 项目简介

鼠标助手标准版是一款功能强大的Windows桌面工具，用于自动化鼠标点击操作。它允许用户选择目标窗口，定义多个鼠标点击事件，并设置时间间隔，从而实现鼠标操作的自动化。
![鼠标助手](https://s2.loli.net/2026/01/22/W9oRKfUQBixSVHy.png)

## 主要功能

- **窗口选择**：轻松选择目标应用程序窗口
- **多任务配置**：支持配置最多10个不同的鼠标点击任务
- **丰富的点击类型**：支持左键单击/双击、右键单击/双击、滚轮单击/双击
- **自定义点击位置**：通过可视化方式选择窗口内的点击位置
- **灵活的时间间隔**：可为每个任务设置固定或随机时间间隔
- **全局快捷键**：支持通过快捷键控制开始/停止操作
- **窗口置顶**：可选择保持目标窗口置顶激活
- **设置导入/导出**：支持保存和加载配置文件，方便在不同场景间切换
- **操作状态显示**：实时显示当前操作状态

## 技术实现

- **开发语言**：C++
- **GUI框架**：Qt 5.15.2
- **构建工具**：qmake
- **系统依赖**：Windows API
- **核心技术**：
  - Windows钩子实现全局热键
  - Windows消息发送实现鼠标事件模拟
  - Qt定时器实现定时操作
  - Qt配置系统实现设置保存/加载

## 使用方法

### 基本操作

1. **选择目标窗口**：
   - 点击"选择窗口"按钮
   - 点击目标应用程序窗口
   - 窗口标题将显示在界面上

2. **配置鼠标点击任务**：
   - 勾选要启用的任务
   - 选择点击类型（左键单击、右键双击等）
   - 点击位置选择按钮，然后点击目标窗口中的位置
   - 设置时间间隔（毫秒）和随机范围

3. **开始/停止操作**：
   - 点击"开始"按钮或使用配置的快捷键开始操作
   - 点击"停止"按钮或使用配置的快捷键停止操作

### 高级功能

- **设置窗口置顶**：勾选"保持选择窗口激活置顶"选项
- **导入/导出设置**：使用工具栏中的导入/导出按钮保存和加载配置
- **快捷键设置**：在界面底部选择合适的快捷键

## 注意事项

- 请在选择点击位置后，不要修改目标窗口大小，否则将导致点击位置不准确
- 某些游戏或应用程序可能会检测并阻止自动化操作
- 使用时请注意遵守相关软件的使用条款

## 常见问题

### Q: 为什么点击位置不准确？
A: 请确保在选择点击位置后没有调整目标窗口的大小或位置，因为点击位置是基于窗口客户区坐标计算的。

### Q: 为什么快捷键没有响应？
A: 请确保选择的快捷键没有与系统或其他应用程序冲突，建议使用Home、End、Page Up等较少使用的按键。

### Q: 为什么无法选择某些窗口？
A: 某些特殊窗口（如UWP应用、系统级窗口）可能无法通过常规方式选择，这是Windows API的限制。
